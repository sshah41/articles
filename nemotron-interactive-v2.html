The 52-Layer Hybrid Stack

Instead of paying the quadratic Attention tax at every layer, Nemotron 3 Super routes processing through three distinct architectures. Interact with the stack below.

Mamba-2 Layers 23 Layers

State-space models operating in linear time. They handle the bulk of sequence processing with a constant-size recurrent state, contributing zero to the KV Cache.

LatentMoE Layers 23 Layers

Highly efficient feed-forward blocks. Projects tokens into a latent space before routing across 128 experts, activating top-6 routed + 2 shared experts.

GQA Attention 6 Layers

Used sparsely as "global anchors" for precise long-range retrieval. By limiting Attention to just 6 layers, memory overhead is drastically minimized.

Core Innovations

Built from the ground up for the Blackwell era, integrating three major structural leaps.

⚡

LatentMoE

Before routing tokens to 128 specialized experts, representations are compressed into a latent space. This allows activating 8 experts per token at the computational cost of far fewer in standard MoE models.

🚀

Multi-Token Prediction

Trained natively to predict multiple future tokens simultaneously. This allows for integrated speculative decoding, yielding up to 3× faster generation speeds without accuracy loss on long reasoning chains.

🔬

NVFP4 Native

Pretrained on 25 Trillion tokens natively in NVIDIA's 4-bit floating-point format from the very first gradient update. The weights learn the quantization space, delivering full-precision accuracy at 4-bit speeds.
